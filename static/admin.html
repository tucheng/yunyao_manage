温度范围配置

前台作品搜索的“低温 / 中温 / 高温”等范围来自这里。编码用于接口参数，保存后不要随意改已有编码。
编码名称最低℃最高℃说明操作

颜色范围配置

颜色名来自发布作品时取色后写入的 color_name 映射结果。多个颜色名用中文逗号或英文逗号分隔，如：天青、粉青、梅子青。
编码名称包含颜色名说明操作

验证码发送配置

控制前台注册、验证码登录和获取验证码时允许填写哪类账号。前期只做短信时请选择“仅短信验证”。
开发调试模式会把验证码返回给前端；上线应切换到邮箱或短信。

邮箱 SMTP

可用变量：{{code}}、{{email}}

短信 HTTP 网关

可用变量：{{code}}、{{phone}}。如果模板是 JSON，会按 JSON 发送。
