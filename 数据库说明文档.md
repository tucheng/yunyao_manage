# 云窑系统 - 数据库说明文档

> 数据库：`yunyao`（MySQL）
> 编码：utf8mb4

---

## 表目录

| # | 表名 | 所属模块 | 简要说明 |
|---|------|----------|----------|
| 1 | `users` | 用户与权限 | 用户账号、微信绑定、等级、余额 |
| 2 | `user_levels` | 用户与权限 | 用户等级定义及发布权限 |
| 3 | `user_settings` | 用户与权限 | 用户默认材料/窑炉/温度偏好 |
| 4 | `recipes` | 配方系统 | 配方主表：标题、描述、分类、价格、釉色 |
| 5 | `recipe_ingredients` | 配方系统 | 配方配料明细：原料、用量、排序 |
| 6 | `recipe_views` | 配方系统 | 配方浏览记录 |
| 7 | `recipe_sequences` | 配方系统 | 配方编号生成器（字母+计数器） |
| 8 | `works` | 作品系统 | 烧制作品：图片、描述、釉色、属性 |
| 9 | `work_comments` | 作品系统 | 作品评论（支持嵌套） |
| 10 | `work_attribute_options` | 作品系统 | 作品可选属性配置（窑/温度/釉面等） |
| 11 | `purchases` | 交易评价 | 配方/作品购买交易记录 |
| 12 | `reviews` | 交易评价 | 评价及回复（含烧制参数） |
| 13 | `favorites` | 社交互动 | 用户收藏（配方/作品） |
| 14 | `likes` | 社交互动 | 用户点赞（配方/作品） |
| 15 | `follows` | 社交互动 | 用户关注关系 |
| 16 | `messages` | 社交互动 | 站内私信 |
| 17 | `notifications` | 社交互动 | 系统通知（评论/关注/点赞/收藏） |
| 18 | `complaints` | 社交互动 | 用户投诉/反馈 |
| 19 | `user_materials` | 材料库存 | 用户材料库存/待购清单 |
| 20 | `ceramic_materials` | 釉料知识 | 陶瓷原材料清单（分子式、分子量） |
| 21 | `glossary_terms` | 釉料知识 | 陶瓷术语定义 |
| 22 | `temperature_cones` | 釉料知识 | 奥尔顿测温锥温度对照 |
| 23 | `firing_curves` | 釉料知识 | 烧制曲线模板（多段升温） |
| 24 | `body_materials` | 釉料知识 | 坯体料类型 |
| 25 | `glazy_materials` | 海外材料 | Glazy 海外材料数据（氧化物含量） |
| 26 | `app_settings` | 系统配置 | 应用配置键值（SMTP/搜索范围等） |
| 27 | `wallet_transactions` | 系统配置 | 钱包充值/提现/消费记录 |
| 28 | `to_be_fired` | 待烧队列 | 用户待烧配方队列（备忘/状态） |

## 一、用户与权限

### 1. `users` 用户表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 用户ID |
| username | VARCHAR(100) UNIQUE | 用户名 |
| openid | VARCHAR(100) UNIQUE | 微信 openid（登录凭证） |
| email | VARCHAR(100) UNIQUE | 邮箱 |
| phone | VARCHAR(20) UNIQUE | 手机号 |
| password_hash | VARCHAR(200) | 密码哈希 |
| nickname | VARCHAR(50) | 昵称 |
| avatar | VARCHAR(200) | 头像URL |
| bio | VARCHAR(500) | 个人简介 |
| gender | VARCHAR(10) | 性别 |
| birthday | VARCHAR(20) | 生日 |
| location | VARCHAR(100) | 所在地 |
| balance | FLOAT | 余额 |
| trust_score | FLOAT | 信任分（0-100） |
| level_id | INTEGER FK→user_levels.id | 用户等级 |
| is_muted | BOOLEAN | 是否禁言 |
| is_admin | BOOLEAN | 是否管理员 |
| created_at | DATETIME | 注册时间 |

### 2. `user_levels` 用户等级表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 等级ID |
| name | VARCHAR(50) | 等级名称 |
| can_publish_paid | BOOLEAN | 能否发布付费内容 |
| max_paid_recipes | INTEGER | 付费配方上限 |
| max_free_recipes | INTEGER | 免费配方上限 |
| max_works | INTEGER | 作品上限 |
| description | VARCHAR(200) | 描述 |
| sort_order | INTEGER | 排序 |

### 3. `user_settings` 用户默认设置

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | INTEGER PK FK→users.id | 用户ID |
| materials | TEXT(JSON) | 默认材料列表 |
| kiln_types | TEXT(JSON) | 默认窑类型 |
| temperatures | TEXT(JSON) | 默认温度 |
| firing_curve_id | INTEGER FK→firing_curves.id | 默认烧制曲线 |
| body_material | VARCHAR(50) | 默认坯体料 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 云窑后台管理（端口 8003）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /admin/glazy-materials?page=&page_size=&q= | 海外材料列表（需token） |
| GET | /admin/glazy-materials/{id} | 海外材料详情 |
| GET | /admin/users?page=&page_size=&q= | 用户管理列表 |
| GET | /admin/stats | 系统概览统计 |

| 方法 | 路径 | 说明 |
